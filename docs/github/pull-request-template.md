# Mẫu Pull Request

Dùng nội dung dưới đây khi mở PR trên GitHub.

---

## Mô tả

Tóm tắt thay đổi và issue được fix (nếu có). Nêu động lực và ngữ cảnh.

Fixes # (issue)

## Loại thay đổi

- [ ] Bug fix (không breaking)
- [ ] Tính năng mới (không breaking)
- [ ] Breaking change
- [ ] Cập nhật tài liệu

## Checklist

- [ ] Code tuân thủ style guide
- [ ] Đã self-review
- [ ] Đã cập nhật tài liệu trong `docs/`
- [ ] Không tạo warning mới
- [ ] Đã thêm test chứng minh fix/feature
- [ ] Test cũ và mới pass locally
