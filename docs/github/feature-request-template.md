# Mẫu đề xuất tính năng (Feature Request)

Dùng nội dung dưới đây khi mở issue trên GitHub. Tiêu đề đề xuất: `feat: mo ta ngan`.

---

**Vấn đề liên quan**

Mô tả vấn đề bạn gặp. Ví dụ: "Tôi luôn gặp khó khăn khi [...]"

**Giải pháp mong muốn**

Mô tả rõ ràng điều bạn muốn xảy ra.

**Giải pháp thay thế đã cân nhắc**

Mô tả các phương án khác bạn đã xem xét.

**Ngữ cảnh thêm**

Screenshot, use case, hoặc thông tin bổ sung.
