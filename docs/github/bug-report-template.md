# Mẫu báo lỗi (Bug Report)

Dùng nội dung dưới đây khi mở issue trên GitHub. Tiêu đề đề xuất: `bug: mo ta ngan`.

---

**Mô tả lỗi**

Mô tả ngắn gọn, rõ ràng về lỗi.

**Cách tái hiện**

1. Import '...'
2. Gõ '...'
3. Quan sát lỗi

**Hành vi mong đợi**

Mô tả rõ điều bạn mong đợi xảy ra.

**Môi trường**

- OS: [ví dụ Windows, macOS]
- Trình duyệt: [ví dụ Chrome, Firefox]
- Phiên bản gotiengviet: [ví dụ 1.0.0]
- Framework: [ví dụ React, Vue, không dùng]

**Ngữ cảnh thêm**

Thông tin bổ sung, screenshot, hoặc đoạn code liên quan.
